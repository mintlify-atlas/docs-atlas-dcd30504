The UI app is a modern single-page application (SPA) built with **Vite** and **React 19**, providing a lightning-fast development experience and production-ready build.

## Architecture

The frontend is built as a client-side SPA with file-based routing using **React Router** v7. It leverages modern React patterns including hooks, context, and server state management with TanStack Query.

## Development

### Running the Dev Server

```bash
# Start UI app only
nx dev ui

# Or start all apps
pnpm dev
```

The dev server runs on **http://localhost:5173** with hot module replacement (HMR).

### Environment Variables

Create `.env` file in the project root:

```bash title=".env"
VITE_API_URL="http://localhost:3001"
VITE_PAYMENT_PROVIDER="polar"  # or "mercadopago"
VITE_MERCADO_PAGO_PUBLIC_KEY="your-public-key"
```

<Note>
  Environment variables must be prefixed with `VITE_` to be exposed to the client.
</Note>

### Building for Production

```bash
# Build UI app
nx build ui

# Preview production build
nx preview ui
```

The build output is in `apps/ui/dist/` and can be deployed to any static hosting service.

## State Management

### Server State (TanStack Query)

```typescript
import { useQuery } from "@tanstack/react-query"

function UserProfile() {
  const { data, isLoading } = useQuery({
    queryKey: ["user"],
    queryFn: () => fetch("/api/user").then(r => r.json()),
    staleTime: 60_000, // 1 minute
  })

  if (isLoading) return <div>Loading...</div>
  return <div>{data.name}</div>
}
```

### URL State (nuqs)

```typescript
import { useQueryState } from "@beztack/state"

function SearchPage() {
  const [search, setSearch] = useQueryState("q")
  
  return (
    <input
      value={search ?? ""}
      onChange={e => setSearch(e.target.value)}
    />
  )
}
```
