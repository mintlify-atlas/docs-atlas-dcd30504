The `@beztack/db` package provides the complete database schema, TypeScript types, and Drizzle ORM client for Beztack applications.

## Installation

```bash
pnpm add @beztack/db
```

## Exports

### db

Drizzle ORM database client instance configured for PostgreSQL.

<ParamField path="db" type="DrizzleClient">
  Pre-configured Drizzle client connected to PostgreSQL using `DATABASE_URL` from environment variables.
</ParamField>

```typescript
import { db } from "@beztack/db"

const users = await db.select().from(user)
```

### schema

Complete database schema object containing all table definitions.

<ParamField path="schema" type="Schema">
  Object containing all Drizzle table definitions for Better Auth adapter and application tables.
</ParamField>

```typescript
import { schema } from "@beztack/db"

// Access individual tables
schema.user
schema.session
schema.organization
```

## Usage Examples

### Querying Data

```typescript
import { db, user, organization } from "@beztack/db"
import { eq } from "drizzle-orm"

// Select all users
const users = await db.select().from(user)

// Select with WHERE clause
const john = await db.select().from(user).where(eq(user.email, "john@example.com"))

// Select specific columns
const userNames = await db.select({ name: user.name, email: user.email }).from(user)
```

### Inserting Data

```typescript
import { db, user } from "@beztack/db"

const newUser = await db
  .insert(user)
  .values({
    id: "user_123",
    name: "John Doe",
    email: "john@example.com",
    emailVerified: false,
  })
  .returning()
```

### Updating Data

```typescript
import { db, user } from "@beztack/db"
import { eq } from "drizzle-orm"

const updated = await db
  .update(user)
  .set({ emailVerified: true })
  .where(eq(user.id, "user_123"))
  .returning()
```

### Joining Tables

```typescript
import { db, user, member, organization } from "@beztack/db"
import { eq } from "drizzle-orm"

const userOrgs = await db
  .select({
    user: user,
    organization: organization,
    role: member.role,
  })
  .from(user)
  .innerJoin(member, eq(member.userId, user.id))
  .innerJoin(organization, eq(organization.id, member.organizationId))
  .where(eq(user.id, "user_123"))
```

### Using in API Routes

```typescript
import { defineEventHandler } from "h3"
import { db, user } from "@beztack/db"
import { eq } from "drizzle-orm"

export default defineEventHandler(async (event) => {
  const { email } = await readBody(event)

  const existingUser = await db.select().from(user).where(eq(user.email, email))

  if (existingUser.length > 0) {
    throw createError({ statusCode: 409, message: "User already exists" })
  }

  const newUser = await db.insert(user).values({ email }).returning()

  return newUser[0]
})
```

## TypeScript Types

Drizzle automatically generates TypeScript types from the schema:

```typescript
import type { InferSelectModel, InferInsertModel } from "drizzle-orm"
import { user, organization } from "@beztack/db"

// Infer select type (read from database)
type User = InferSelectModel<typeof user>

// Infer insert type (write to database)
type NewUser = InferInsertModel<typeof user>

// Use in function signatures
function processUser(user: User): void {
  console.log(user.email)
}

function createUser(data: NewUser): Promise<User> {
  // ...
}
```
