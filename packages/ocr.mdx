# OCR Package

The `@beztack/ocr` package provides optical character recognition (OCR) capabilities using Tesseract.js, allowing you to extract text from images.

## Installation

The OCR package is optional. Add it during project creation:

```bash Terminal
pnpm dlx beztack init --modules ocr
```

Or install manually:

```bash Terminal
pnpm add @beztack/ocr --filter @beztack/api
```

## Usage Examples

### Basic Text Extraction

```typescript Example
import { extractTextFromImage } from "@beztack/ocr"

const text = await extractTextFromImage("/path/to/document.png")
console.log(text)
// Output: "This is the text from the image"
```

### Multi-language OCR

```typescript Spanish + English
import { extractTextFromImage } from "@beztack/ocr"

const text = await extractTextFromImage(
  "/path/to/bilingual.jpg",
  "eng+spa"
)
console.log(text)
```

### API Route Handler

```typescript Nitro Route
import { extractTextFromImage } from "@beztack/ocr"
import { defineEventHandler, readMultipartFormData, createError } from "h3"
import { writeFile } from "node:fs/promises"
import { join } from "node:path"

export default defineEventHandler(async (event) => {
  const formData = await readMultipartFormData(event)
  const imageFile = formData?.find((field) => field.name === "image")

  if (!imageFile) {
    throw createError({ statusCode: 400, message: "No image provided" })
  }

  // Save uploaded file temporarily
  const tempPath = join("/tmp", `ocr-${Date.now()}.png`)
  await writeFile(tempPath, imageFile.data)

  try {
    const text = await extractTextFromImage(tempPath, "eng")
    return { text }
  } catch (error) {
    throw createError({
      statusCode: 500,
      message: "OCR processing failed",
    })
  }
})
```

### Processing Remote Images

```typescript URL Input
import { extractTextFromImage } from "@beztack/ocr"

const imageUrl = "https://example.com/receipt.jpg"
const text = await extractTextFromImage(imageUrl, "eng")
console.log(text)
```

### Document Processing

```typescript Invoice OCR
import { extractTextFromImage } from "@beztack/ocr"
import { db, document } from "@beztack/db"

async function processInvoice(imagePath: string) {
  // Extract text
  const rawText = await extractTextFromImage(imagePath, "eng")

  // Parse extracted text (simplified example)
  const invoiceNumber = rawText.match(/Invoice #: (\d+)/)?.[1]
  const totalAmount = rawText.match(/Total: \$(\d+\.\d{2})/)?.[1]

  // Save to database
  await db.insert(document).values({
    id: crypto.randomUUID(),
    type: "invoice",
    content: rawText,
    metadata: JSON.stringify({
      invoiceNumber,
      totalAmount: totalAmount ? Number.parseFloat(totalAmount) : null,
    }),
  })

  return { invoiceNumber, totalAmount }
}
```

## Performance Tips

### Image Preprocessing

For better accuracy, preprocess images before OCR:

```typescript Image Optimization
import sharp from "sharp"
import { extractTextFromImage } from "@beztack/ocr"

async function ocrWithPreprocessing(inputPath: string) {
  const processedPath = "/tmp/processed.png"

  // Enhance image for better OCR
  await sharp(inputPath)
    .grayscale() // Convert to grayscale
    .normalize() // Improve contrast
    .sharpen() // Enhance edges
    .toFile(processedPath)

  return extractTextFromImage(processedPath)
}
```

### Worker Reuse

<Note>
  The current implementation creates and terminates a worker for each request. For high-volume processing, consider creating a worker pool.
</Note>

## Error Handling

```typescript Try-Catch
import { extractTextFromImage } from "@beztack/ocr"
import { defineEventHandler, createError } from "h3"

export default defineEventHandler(async (event) => {
  try {
    const text = await extractTextFromImage("/path/to/image.png", "eng")
    return { success: true, text }
  } catch (error) {
    console.error("OCR failed:", error)
    throw createError({
      statusCode: 500,
      message: error instanceof Error ? error.message : "OCR processing failed",
    })
  }
})
```

## Source Code

```typescript packages/ocr/index.ts:1-12
import { createWorker } from "tesseract.js"

export const extractTextFromImage = async (
  imagePath: string,
  lang?: string
) => {
  const worker = await createWorker(lang)
  const ret = await worker.recognize(imagePath)
  await worker.terminate()
  return ret.data.text
}
```
