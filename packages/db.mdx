# Database Package

The `@beztack/db` package provides a type-safe database client using Drizzle ORM with PostgreSQL. It includes pre-configured tables for authentication, organizations, teams, and payment providers.

## Installation

The database package is required and included by default:

```bash Terminal
pnpm add @beztack/db --filter @beztack/api
```

## Usage Examples

### Query Users

```typescript Example
import { db, user } from "@beztack/db"
import { eq } from "drizzle-orm"

// Find user by email
const users = await db
  .select()
  .from(user)
  .where(eq(user.email, "user@example.com"))

const foundUser = users[0]
```

### Insert Organization

```typescript Create
import { db, organization } from "@beztack/db"

const [newOrg] = await db
  .insert(organization)
  .values({
    id: crypto.randomUUID(),
    name: "Acme Corp",
    slug: "acme-corp",
  })
  .returning()

console.log(newOrg.id)
```

### Join Tables

```typescript Relations
import { db, organization, member, user } from "@beztack/db"
import { eq } from "drizzle-orm"

// Get organization members with user details
const members = await db
  .select({
    userId: user.id,
    userName: user.name,
    userEmail: user.email,
    role: member.role,
    orgName: organization.name,
  })
  .from(member)
  .innerJoin(user, eq(member.userId, user.id))
  .innerJoin(organization, eq(member.organizationId, organization.id))
  .where(eq(organization.slug, "acme-corp"))
```

### Transactions

```typescript Atomic Operations
import { db, user, organization, member } from "@beztack/db"

await db.transaction(async (tx) => {
  // Create organization
  const [org] = await tx
    .insert(organization)
    .values({
      id: crypto.randomUUID(),
      name: "New Company",
      slug: "new-company",
    })
    .returning()

  // Add user as owner
  await tx.insert(member).values({
    id: crypto.randomUUID(),
    organizationId: org.id,
    userId: "user-id",
    role: "owner",
  })
})
```

## Schema Definition

### User Table

```typescript Schema (packages/db/src/schema.ts:14-35)
export const user = pgTable("user", {
  id: text("id").primaryKey(),
  name: text("name").notNull(),
  email: text("email").notNull().unique(),
  emailVerified: boolean("email_verified").default(false).notNull(),
  image: text("image"),
  createdAt: timestamp("created_at").defaultNow().notNull(),
  updatedAt: timestamp("updated_at")
    .defaultNow()
    .$onUpdate(() => new Date())
    .notNull(),
  twoFactorEnabled: boolean("two_factor_enabled").default(false),
  role: text("role"),
  banned: boolean("banned").default(false),
  banReason: text("ban_reason"),
  banExpires: timestamp("ban_expires"),
  // Subscription cache (denormalized)
  subscriptionTier: text("subscription_tier"),
  subscriptionStatus: text("subscription_status"),
  subscriptionId: text("subscription_id"),
  subscriptionValidUntil: timestamp("subscription_valid_until"),
})
```

## Database Commands

The package includes scripts for database management:

### Generate and Apply Migrations

```bash Terminal
pnpm migrate
```

Generates migration files from schema changes and applies them to the database.

### Push Schema Directly

```bash Terminal
pnpm push
```

Pushes schema changes directly to the database without creating migration files (useful for development).

<Warning>
  Use `pnpm push` only in development. For production, always use migrations with `pnpm migrate`.
</Warning>

### Generate Auth Schema

```bash Terminal
pnpm generate-auth-schema
```

Regenerates Better Auth schema types from auth configuration.

## Configuration

Database connection is configured via environment variables:

```typescript Source (packages/db/src/client.ts:1-8)
import { drizzle } from "drizzle-orm/postgres-js"
import postgres from "postgres"
import { env } from "@beztack/env/api"

const connectionString = env.DATABASE_URL
const client = postgres(connectionString)
export const db = drizzle({ client })
```

Required environment variable:

```bash .env
DATABASE_URL=postgresql://user:password@localhost:5432/dbname
```

## Type Safety

Drizzle provides full TypeScript type inference:

```typescript Type Inference
import { db, user } from "@beztack/db"

const users = await db.select().from(user)
// users: Array<{
//   id: string
//   name: string
//   email: string
//   emailVerified: boolean
//   ...
// }>

const userEmail: string = users[0].email // ✅ Type-safe
```
