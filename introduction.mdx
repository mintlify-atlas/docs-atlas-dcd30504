# Introduction to Beztack

**You want to ship. You need this.**

Beztack is a modern TypeScript monorepo starter designed to help you ship faster. Stop wasting time on boilerplate configuration and start building your product with enterprise-grade tooling from day one.

## What Problem Does Beztack Solve?

Building a modern web application requires integrating dozens of tools and libraries. Most developers spend weeks configuring:

- Build tools and development environments
- Authentication with 2FA and organizations
- Database schemas and type-safe queries
- Payment integrations and webhooks
- Email templates and deliverability
- Code quality and formatting tools
- Monorepo tooling and workspace management

Beztack eliminates this setup time by providing a **production-ready foundation** with all these features pre-configured and working together seamlessly.

## Architecture

Beztack uses a monorepo architecture powered by **NX** with three specialized apps:

```
apps/
├── ui/          # Frontend SPA (Vite + React)
├── api/         # Backend Server (Nitro)
└── docs/        # Landing & Documentation (Next.js + Fumadocs)

packages/
├── ai/          # AI SDK integration (Vercel AI + Amazon Bedrock)
├── cli/         # CLI tools for project scaffolding
├── email/       # Email templates (React Email)
├── env/         # Environment validation (T3 Env)
├── ocr/         # OCR utilities
└── state/       # URL state management (nuqs)
```

### Frontend (`apps/ui`)

- **Vite** for lightning-fast HMR and optimized builds
- **React 19** with modern concurrent features
- **shadcn/ui** components built on Radix UI primitives
- **TanStack Router** for type-safe client-side routing
- **TanStack Query** for server state management and caching
- **Tailwind CSS v4** for utility-first styling

### Backend (`apps/api`)

- **Nitro** (h3) for edge-ready, portable server architecture
- **better-auth** for authentication with 2FA, Organizations, and Admin roles
- **Drizzle ORM** for type-safe database queries with PostgreSQL
- **Polar** integration for payments and subscriptions
- File-based routing with auto-imports
- Full TypeScript support with Zod validation

### Documentation (`apps/docs`)

- **Next.js 15** with App Router for optimal performance
- **Fumadocs** for beautiful MDX documentation
- Server-side rendering and static generation
- SEO optimized out of the box

## Use Cases

- **SaaS Applications** - Authentication, subscriptions, and user management built-in
- **AI-Powered Apps** - Vercel AI SDK integrated with streaming and tool calling
- **E-commerce Platforms** - Payment processing with Polar or Mercado Pago
